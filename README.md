# 리눅스 명령어 조사

## 1. top

top은 현재 실행 중인 프로세스, CPU, 메모리 사용량을 실시간으로 확인하는 명령어이다.

### 주요 옵션

- `-d` : 갱신 주기 설정
- `-n` : 실행 횟수 지정
- `-u` : 특정 사용자 프로세스 조회

### 사용 예시

```bash
top
```

## 2. ps

ps는 현재 실행 중인 프로세스의 상태를 확인하는 명령어이다.

### 주요 옵션

- `-e` : 모든 프로세스 출력
- `-f` : 상세 정보 출력
- `-u` : 특정 사용자 프로세스 출력

### 사용 예시

```bash
ps -ef
```

## 3. jobs

jobs는 현재 쉘에서 실행 중인 작업(Job)을 확인하는 명령어이다.

### 주요 옵션

- `-l` : PID 포함 출력
- `-n` : 상태가 변경된 작업만 출력
- `-p` : PID만 출력

### 사용 예시

```bash
jobs
```

## 4. kill

kill은 특정 프로세스를 종료할 때 사용하는 명령어이다.

### 주요 옵션

- `-9` : 강제 종료
- `-15` : 정상 종료
- `-l` : 시그널 목록 출력

### 사용 예시

```bash
kill -9 1234
```
