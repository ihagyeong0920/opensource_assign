# Password Strength Checker

## 프로그램 소개

Password Strength Checker는 사용자가 입력한 비밀번호의 강도를 분석하는 프로그램이다.

비밀번호의 길이와 대문자, 소문자, 숫자의 포함 여부를 검사하여 비밀번호 강도를 평가한다.

---

## 주요 기능

* 비밀번호 길이 검사 (8자 이상)
* 대문자 포함 여부 검사
* 소문자 포함 여부 검사
* 숫자 포함 여부 검사
* 비밀번호 강도 평가

---

## 실행 방법

`python password_checker.py`

---

## 실행 예시

### 입력

Abcd1234

### 출력

비밀번호 강도 : 강함

### 입력

abc

### 출력

8자 이상 입력하세요.

대문자가 포함되어 있지 않습니다.

숫자가 포함되어 있지 않습니다.

비밀번호 강도 : 약함

---

## 개발 목적

사용자가 안전한 비밀번호를 생성할 수 있도록 비밀번호의 강도를 간단하게 확인하기 위해 개발하였다.

---

## 자기 평가

본 프로그램은 비밀번호의 길이와 대문자, 소문자, 숫자의 포함 여부를 검사하여 비밀번호 강도를 평가하도록 구현하였다.

Python의 조건문과 문자열 관련 함수를 활용하여 비교적 간단한 방식으로 구현할 수 있었으며, 사용자에게 비밀번호의 취약한 부분을 알려줄 수 있도록 하였다.

향후에는 특수문자 포함 여부 검사, 비밀번호 점수 세분화, 다양한 보안 규칙 적용 기능을 추가하여 더욱 정확한 비밀번호 강도 평가 프로그램으로 개선할 수 있을 것이다.
